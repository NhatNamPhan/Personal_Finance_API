User Endpoints
POST /users - Tạo user mới
GET /users/{user_id} - Lấy thông tin user theo ID

Account Endpoints
POST /accounts - Tạo tài khoản mới
GET /users/{user_id}/accounts - Lấy danh sách tài khoản của user

Category Endpoints
POST /categories - Tạo danh mục mới (income/expense)
GET /users/{user_id}/categories - Lấy danh sách danh mục của user
GET /users/{user_id}/categories?type=income - Lọc danh mục theo type (income/expense)

Transaction Endpoints
POST /transactions - Thêm giao dịch mới (quan trọng - tự động cập nhật balance)
GET /users/{user_id}/transactions - Lấy lịch sử giao dịch của user
GET /users/{user_id}/transactions?start_date=2024-01-01 - Lọc giao dịch từ ngày
GET /users/{user_id}/transactions?end_date=2024-01-31 - Lọc giao dịch đến ngày
GET /users/{user_id}/transactions?start_date=2024-01-01&end_date=2024-01-31 - Lọc giao dịch theo khoảng thời gian

Budget Endpoints
POST /budgets - Tạo ngân sách mới
GET /users/{user_id}/budgets - Lấy danh sách ngân sách của user
GET /users/{user_id}/budgets?month=2024-01-01 - Lọc ngân sách theo tháng

Analytics Endpoints
GET /users/{user_id}/spending?start_date=2024-01-01&end_date=2024-01-31 - Thống kê chi tiêu theo danh mục
GET /users/{user_id}/budgets/progress?month=2024-01-01 - Tiến độ ngân sách (so sánh budget với thực tế)
GET /users/{user_id}/net-worth - Tổng tài sảng ròng (Net Worth)
